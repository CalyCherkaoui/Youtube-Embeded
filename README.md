# Youtube-Embeded
Building an HTML document that matches the appearance of Youtube.

## Built With
- HTML & CSS

## Usage
```Git
git clone https://github.com/CalyCherkaoui/Youtube-Embeded.git
cd Youtube-Embeded
git checkout FeatureB-css
firefox .
```
